# roadmap-ai

AI-powered personalized learning roadmap generator.

## What it does
Enter any skill or topic and get a structured, AI-generated learning roadmap with resources, milestones, and timelines.

## Tech Stack
- Frontend: Next.js + Tailwind CSS
- AI: Groq LLM API
- Deployment: Vercel
